Standard Death Record Implementation Guide

The Standard Death Record (SDR) Implementation Guide (IG) describes how FHIR can be used to exchange death event information between death certifiers, funeral directors and death registration systems.

The source material and instructions for generating this IG are available at the following URL: https://github.com/nightingaleproject/fhir-death-record .

Document Status

This is a preliminary version of the IG intended for use in prototyping, interoperability testing, and to provoke discussion. This IG has not yet been balloted.
